# Full Stack Redux Tutorial
http://teropa.info/blog/2015/09/10/full-stack-redux-tutorial.html

## Setup
```
git clone https://github.com/jsfeb26/voting-server.git
cd voting-server
npm install
```

## Running Tests
To run all tests:
```
npm run test
```
Or to watch tests run:
```
npm run test:watch
```

## Running The Application
```
npm run start
```
